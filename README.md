CM Map Downloader & Stitcher
(CM 高清地图下载与拼接工具)这是一个自动化 Python 脚本，用于从切片服务器下载高清地图图片，并自动将其拼接成完整的无缝大图。

✨ 功能特点
自动解析链接：只需复制任意一张切片的完整 URL，脚本自动提取 Base URL 和 SAS Token。
自定义范围：支持指定下载的行列范围（例如 6-26）。
断点续传：自动跳过已下载的图片，节省时间。
智能拼接：自动识别行列坐标（X-Y 格式），生成完整的 PNG 大图。

🛠️ 环境要求
Python 3.x依赖库：requests, Pillow
📦 安装
克隆或下载本仓库代码。
安装必要的 Python 库：pip install requests pillow

🚀 使用方法
运行脚本：python cmk-map.py
按照提示输入信息：
链接：在浏览器开发者模式中找到的任意一张切片图片的完整链接。
范围：输入想要下载的最大列号和行号（格式如 6-26）。
文件名：输出大图的文件名。
结果：下载的小图保存在 downloaded_tiles/ 目录。拼接好的大图保存在当前根目录。
